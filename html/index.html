CONTAMOS CON DOS PLANES QUE BRINDAN SOLUCIONES A TU NEGOCIO

BASIC - LEASING

Plan basico el cual consta de arrendar una cantidad limitada de equipos.

  • ✓ Renovación de equipos cada 24 meses.
  • ✓ Soporte técnico en horario laboral (lunes a viernes, 9am-6pm).
  • ✓ Gestión de garantías incluida.
  • ✓ Mantenimiento preventivo 1 vez al año.
  • ✓ Incluye consultoría personalizada TI
  • ✓ Soporte 24/7.
  • ✓ Reposición inmediata de equipos (tiempo estándar 5-7 días).
  • ✓ Incluye licencias de software.

SMART - LEASING

La solución avanzada para organizaciones en expansión.

  • ✓ Renovación de equipos cada 12-18 meses.
  • ✓ Soporte técnico 24/7 (remoto y presencial).
  • ✓ Consultoría en optimización de costos TI.
  • ✓ Reemplazo inmediato en fallas críticas.
  • ✓ Mantenimiento preventivo cada 6 meses.
  • ✓ Gestión de licencias de software (Office, antivirus, etc.).
  • ✓ Reportes de uso y métricas de desempeño.
